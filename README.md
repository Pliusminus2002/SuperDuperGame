SuperDuperGame is a browser-based 3D game built with Three.js.
It was originally created as a personalized interactive experience for a friend and later refined into a technical portfolio project demonstrating real-time 3D development in the browser.

The project focuses on building a fully interactive 3D environment using WebGL through Three.js, handling 3D models, textures, lighting, and background audio while deploying and maintaining a live version online.
